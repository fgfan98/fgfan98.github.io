发表于2019-12-13|Java
|阅读量:

AOP（Aspect Oriented Programming）称为面向切面编程，在程序开发中主要用来解决一些系统层面上的问题，比如日志，事务，权限等待，Struts2的拦截器设计就是基于AOP的思想。

定义一个Proxy的子类，实现需要代理的类所实现的接口，在重写方法时插入切片。
